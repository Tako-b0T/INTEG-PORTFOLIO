HTML Links

Anchor tag: <a href="URL">link</a>

Absolute link: <a href="URL" target="_blank">link</a>

Relative link: <a href="Page1.html">Page1</a>

Internal link: <a href="#top">jump to top</a>

Image Format Overview

  • GIF: Good for simple graphics, supports animation
  • JPEG: Great for photos
  • PNG: Supports transparency and lossless compression
  • WebP: Modern format with high compression, transparency
