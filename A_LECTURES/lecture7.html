Lecture 7: Bootstrap

What is Bootstrap?

  • Bootstrap is a free, open-source HTML, CSS, and JavaScript framework for responsive web design.
  • Originally developed by Twitter engineers Mark Otto and Jacob Thornton.
  • Enables mobile-first, cross-browser responsive designs.

Why Use Bootstrap?

  • Mobile-first design philosophy
  • Compatible with all major browsers
  • Fast development with built-in components and utilities
  • Clean, uniform styling and flexible customization

Bootstrap Package Includes

  • Scaffolding: basic structure and grid system
  • CSS: global styles and components
  • Components: navbars, alerts, modals, etc.
  • JavaScript Plugins: custom jQuery plugins
  • Customization options

Containers

  • .container: Fixed-width responsive container
  • .container-fluid: Full-width container
  • Padding: .p-3, .pt-3, etc.
  • Background and border examples with .bg-* and .border

Grid System

  • 12-column grid using flexbox
  • Classes: .col-, .col-sm-, .col-md-, .col-lg-, .col-xl-
  • Responsive design: columns stack on smaller screens
  • Auto layout using .col
