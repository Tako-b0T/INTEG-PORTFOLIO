Lecture 4: Lists, Links, and Images

1. HTML Lists

  • Ordered List (<ol>): Numbered lists ideal for instructions or hierarchical items.
  • Unordered List (<ul>): Bulleted lists for unordered content.
  • Description List (<dl>): Lists of terms with descriptions.
  • Nested Lists: Lists within lists for categorized data.
  • CSS Styling: Use list-style-type to change marker styles like upper-roman, square, none, etc.

2. Links / Hyperlinks

  • <a> Tag: Used to create hyperlinks; href defines the target URL.
  • Absolute URL: Links to external websites. (e.g., http://example.com)
  • Relative URL: Links to local site pages. (e.g., pages/about.html)
  • Anchor Links: Use #id to jump within the same page.
  • Link Targets: Control link behavior with target="_blank", _self, etc.

3. HTML Links as Images

  • Clickable Images: Wrap <img> in <a> tags to make images links.
  • Example: <a href="page.html"><img src="img.jpg"></a>

4. Image Embedding

  • <img> Tag: Defines images. Use src for path and alt for alternative text.
  • Size Control: Use style to set width and height.
  • Floating Images: Use CSS float: left/right; for text wrapping.
  • Folder Structure: Use folder names in src if images are stored in subfolders.
  • External Images: Use full URL in src to embed from other websites.

5. Image Formats

  • GIF: Best for simple graphics and animations. Max 256 colors. Supports transparency and interlacing.
  • JPEG: Best for photos. Supports thousands of colors. Uses lossy compression.
  • PNG: Supports transparency, true-color. Good for lossless quality.
  • WebP: Googleâ€™s format with lossy/lossless compression and transparency support. Highly efficient.

6. Image Maps

  • Image Map: Defines clickable areas (hot spots) within an image using <map> and <area>.
  • Shapes: rect, circle, poly, and default are used to define clickable zones.
  • Attributes: Use coords, href, title, and alt for functionality and accessibility.

7. Background Images

  • CSS Property: Use background-image: url('filename.jpg') in inline styles or stylesheets.
  • Application: Can be applied to <body> or any other container.
