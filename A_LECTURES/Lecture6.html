Lecture 6: Forms and Multimedia

1. Introduction to Web Forms

  • Web forms allow users to input and submit data through a webpage.
  • Controls include: text inputs, radio buttons, checkboxes, select lists, text areas, and buttons.
  • Widgets include: sliders, spinners, calendars, and color pickers.

2. Basic Form Elements

  • <form>: Wraps all form elements; attributes include action, method, and target.
  • <input>: Used for various input types such as text, password, radio, checkbox, etc.
  • <label>: Associates text with form controls for accessibility and usability.

3. Form Submission

  • GET: Appends form data to the URL (suitable for short, non-sensitive data).
  • POST: Sends data in the body of the request (better for large or sensitive data).
  • Target: Defines where the response opens (e.g., _blank for a new tab).

4. Input Types and Attributes

  • Text, Password, Radio, Checkbox, Submit, Reset, Button, Color
  • HTML5 types include: date, email, number, range, search, tel, time, url, week
  • Attributes include:
    • value – default input value
    • readonly – input cannot be edited
    • disabled – input is unclickable
    • size and maxlength – control text field width and limit
    • Others: placeholder, required, pattern, autofocus, etc.

5. Additional Form Elements

  • <select> and <option>: Create drop-down lists with optional multiple and size.
  • <textarea>: Multi-line text input field with rows and cols.
  • <fieldset> and <legend>: Group and label related fields in a form.

6. Form Design Tips

  • Group related inputs logically using fieldsets.
  • Use labels and placeholder text to guide the user.
  • Optimize for both desktop and limited screen displays.
  • Ensure logical flow and user-friendly layout.

7. Introduction to Multimedia

  • Multimedia includes: audio, video, animations, and interactive content.
  • Common formats: .mp3, .mp4, .ogg, .webm, .avi

8. Embedding Audio

  • Use the <audio> tag with controls and <source> for fallback options.
  • Example: 
    <audio controls>
  <source src="file.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

9. Embedding Video

  • Use the <video> tag with controls, width, and height attributes.
  • Include multiple <source> tags for broader compatibility.
  • Example: 
    <video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

10. Plugins and Embed Elements

  • <object>: Embeds external content like PDFs, Flash, or other HTML.
  • <embed>: Similar to <object>, used for legacy plugins and media.
  • Examples: 
    <object width="400" height="200" data="file.swf"></object>
<embed src="file.swf" width="400" height="200">
