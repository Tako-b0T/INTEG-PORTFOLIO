Lecture 3: Lists and Tables

1. HTML Styles

  • Style Attribute: Used to add inline styles (color, font, size).
  • Syntax: <tagname style="property:value;">

2. Styling Examples

  • Background Color: <body style="background-color:DarkTurquoise;">
    <h1 style="background-color:cyan;">
  • Text Color: <p style="color:blue;">
  • Fonts: font-family: verdana; courier;
  • Text Size: font-size: 300%; 30pt;
  • Text Alignment: text-align: center;

3. HTML Tables Overview

  • <table>: Creates a table.
  • <tr>: Table row.
  • <th>: Header cell (bold and centered by default).
  • <td>: Standard data cell.

4. Table Example

<table style="width:100%">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Nicole</td>
    <td>Pascual</td>
    <td>18</td>
  </tr>
</table>

5. Table Styling with CSS

  • Border: table, th, td { border: 1px solid black; }
  • Collapsed Border: border-collapse: collapse;
  • Padding: th, td { padding: 15px; }
  • Text Align: th { text-align: left; }
  • Border Spacing: table { border-spacing: 5px; }

6. Column and Row Spanning

  • Colspan: Merges cells horizontally.
    <th colspan="2">Telephone</th>
  • Rowspan: Merges cells vertically.
    <th rowspan="2">Telephone:</th>
