Lectures

1. Evolution of Web Design

  • First website by Tim Berners-Lee (1990): A simple text-based page explaining the World Wide Web.
  • Formation of W3C in 1994: Created to standardize web technologies and prevent proprietary web languages.
  • Table-based layouts in the 90s: Tables and spacer GIFs allowed basic layout control and visual structure.
  • Flash and its decline: Enabled rich interactive websites, but lacked SEO support and was dropped by Apple in 2010.
  • Rise of CSS and separation of content and design: CSS allowed styling to be separated from HTML, improving maintainability and performance.

2. Web 2.0 and JavaScript

  • Interactive and intelligent web: JavaScript enabled dynamic content, multimedia, and user interactions.
  • Focus on usability and SEO: Designers optimized navigation and content structure to improve search visibility and user experience.
  • Mobile browsing growth and responsive design: Responsive layouts became necessary as mobile usage surpassed desktop by 2016.

3. Modern Design Trends

  • Media Queries and Grid Systems: CSS3 media queries allow flexible layouts based on screen size; grids help structure content visually.
  • Flat Design and Swiss Style: Simplified design approach removing shadows and textures, influenced by mid-century Swiss design principles.
  • Typography and font choices: Font style impacts the website's mood; using combinations of serif, sans-serif, and script helps branding.
  • Color psychology in design: Colors influence emotions and behaviors, and choosing the right palette affects how users perceive the site.

4. Key Components of Good Web Design

  • Grids: Help maintain consistent layout and alignment, improving readability and visual balance.
  • Call to Action (CTA): Prompts user actions like buying or signing up, often styled for visibility and strategically placed.
  • Breadcrumbs: Show users their navigation path on complex websites, improving site orientation.
  • Search Bars: Essential for content-heavy websites; must be easy to find and use for efficiency.
  • Icons: Visual cues that aid navigation and understanding while improving aesthetic appeal.
  • Modals: Pop-ups used for alerts, forms, confirmations, or guidance without leaving the page.

5. Principles of Usability

  • Learnability, Efficiency, Memorability, Error Tolerance, Satisfaction: Key metrics that determine how easy and pleasant a website is to use.
  • Simplicity and Navigability: Users prefer clean, intuitive layouts with easy-to-follow navigation.
  • Accessibility and Responsiveness: Websites should work across devices and load quickly with minimal errors or downtime.
  • Consistency in Design, Content, and Interaction: Predictable behavior and uniform styles make websites more user-friendly and professional.
