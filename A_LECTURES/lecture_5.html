Lecture 5: CSS3

Introduction to CSS3

  • CSS separates content from presentation
  • W3C developed CSS1 (1996), CSS2 (1998), and modular CSS3
  • CSS3 is backward compatible with CSS2.1
  • Over 50 CSS3 modules provide modern styling capabilities

CSS Syntax and Selectors

  • Selector: Targets HTML element
  • Declaration: Property-value pair inside braces
  • Types of selectors: element, id (#id), class (.class), group
  • Comments: /* comment */

Ways to Insert CSS

  • External: Linked using <link> in <head>
  • Internal: CSS within <style> tag
  • Inline: Style inside HTML element using style attribute

Cascading and Color Properties

  • Priority: Inline > Internal > External > User-defined > Browser default
  • Color types: named, HEX, RGB, RGBA, HSL, HSLA
  • background-color, color, border-color

Backgrounds and Borders

  • Backgrounds: color, image, repeat, attachment, position
  • Shorthand: background
  • Borders: style, width, color, radius (for rounded corners)
  • Shorthand: border: 2px solid red;

Box Dimensions

  • width and height: px, %, auto
  • max-width limits element size

CSS Rule Example

  • Selector: p
  • Declaration block: { color: red; text-align: center; }
