Lecture 5: CSS

1. Introduction to CSS3

  • CSS separates content from presentation and ensures consistency across webpages.
  • CSS3 is modular and backward-compatible with CSS2.1.
  • W3C began CSS3 development by splitting it into over 50 modules.
  • CSS allows designers to define layout, colors, fonts, borders, and more.

2. Why CSS Solved a Big Problem

  • Before CSS, HTML had to include styling with tags like <font>, which was inefficient.
  • CSS allows external styling, making it easier to manage large websites.

3. CSS Syntax and Selectors

  • Rule-set: Consists of a selector and declaration block.
  • Selectors:
    • Element selector: targets tags like p, h1
    • ID selector: #idName targets unique elements
    • Class selector: .className targets multiple elements
    • Grouping: Combine selectors like h1, h2, p
  • Comments: Written as /* comment */

4. Inserting CSS

  • External: Linked via <link> in <head>
  • Internal: Placed within a <style> tag in the HTML file
  • Inline: Added directly to HTML tags with the style attribute

5. Cascading Order

  • Priority of rules (highest to lowest):
    1. Inline style
    2. Internal style
    3. External CSS
    4. User-defined settings
    5. Browser default

6. CSS Colors

  • Use color for background, text, and borders.
  • Color formats:
    • Color names: red, blue
    • HEX: #ff0000
    • RGB: rgb(255,0,0)
    • RGBA: rgba(255,0,0,0.5)
    • HSL/HSLA: hsl(120, 100%, 50%)

7. Backgrounds

  • Properties: background-color, background-image, background-repeat, background-position, background-attachment
  • Shorthand: Combine all in background
  • Example: background: #fff url('img.png') no-repeat right top;

8. Borders

  • Styles: dotted, dashed, solid, double, groove, ridge, inset, outset
  • Width: Use px, pt, %, or keywords like thin, thick
  • Color: Specify using name, HEX, RGB, or transparent
  • Individual Sides: e.g., border-left, border-top
  • Rounded borders: Use border-radius to create curves

9. Height and Width

  • Units: px, %, cm, em
  • auto: Let the browser decide
  • max-width: Restricts the maximum allowed width of an element
