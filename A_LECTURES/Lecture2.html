Lecture 2: HTML Structures

1. Tools Professionals Use

  • Computer: Any spec works, but better specs help.
  • Text Editor: Default Notepad or Notepad++.
  • Web Browser: Chrome, Firefox, Edge, etc.
  • Graphic Software: GIMP, Figma, Photoshop.
  • Version Control: GitHub or GitLab to manage files.
  • FTP Client (Optional): Cyberduck, FileZilla.
  • Automation Tools: For repetitive tasks.

2. Planning Your Website

  • Define Purpose: Know the site's goal.
  • Know Your Audience: Design with users in mind.
  • Research Similar Sites: Learn from others.
  • Storyboard First: Map it out before building.

3. Folder Structure

  • index.html: Homepage file.
  • images/: Store graphics here.
  • styles/: For CSS files.
  • scripts/: For JavaScript files.

4. Introduction to HTML

  • HTML: A markup language used to structure web pages.
  • HTML5: Current standard with semantic tags, audio/video, canvas, etc.

5. HTML5 Semantic Tags

  • <header>: Introduces content or sections.
  • <section>: Groups thematic content.
  • <article>: Self-contained content.
  • <nav>: Major navigation links.
  • <aside>: Sidebars and tangential content.
  • <footer>: Footer info for page or section.
  • <figure> and <figcaption>: Images with captions.

6. HTML Concepts

  • Elements: Containers for page content. Ex: <p>Text</p>
  • Tags: Mark up content. Opening and closing tags used.
  • Attributes: Provide additional info. Ex: align="center"
  • Nesting: Elements must be properly nested without overlap.

7. Common Text Tags

  • <h1>–<h6>: Heading levels.
  • <p>: Paragraphs.
  • <br>, <hr>: Line breaks and horizontal lines.
  • <b>, <strong>: Bold and emphasized text.
  • <i>, <em>, <cite>: Italics, emphasis, citations.
  • <code>, <kbd>, <samp>: Monospaced/code text.
  • <u>, <mark>, <del>, <ins>: Underline, highlight, deleted/inserted text.
  • <sup>, <sub>: Superscript and subscript.

8. Formatting and Layout Tips

  • <blockquote>: Indents a block of text.
  • <pre>: Preserves formatting.
  • Text Alignment: Use align="left|center|right|justify".
  • Spacing: Use &nbsp; for extra spaces.

9. HTML Entities

  • &lt;: <
  • &gt;: >
  • &amp;: &
  • &copy;: ©
  • &nbsp;: Non-breaking space
